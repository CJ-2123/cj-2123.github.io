KH Image Tracker

Simple HTML tracker for various item types in KH.

https://cj-2123.github.io/KH2-image-tracker

Features

The user can pick from various item categories

It's setup for other categories of items to be easy to add if needed. Right now, it supports KH2 Keyblades and Recipes and KH3 Ingredients to get Master Chef achievement.

It also has two colors to cycle through when you click on an image.

Future Work

  • Dynamically resize images
    • Currently, images need to be resized manually to fit
  • Reset tracker category when select a new category
    • Currently, selecting a new category does not get rid of the old items. Refreshing the page will.
  • Allow user to select colors
